# PM Internship Scheme Portal (PMIS)

A full-stack web application developed during **Smart India Hackathon (SIH) 2025** aimed at building an intelligent **Internship Recommendation Engine** for the **PM Internship Scheme**.

The platform was designed to help students discover relevant internship opportunities based on their profiles, skills, and interests while simplifying the application and management process for administrators and organizations.

I worked as the **Team Lead** and primarily contributed to the **Full Stack Development** of the project.

## 🚀 Project Overview

The PM Internship Scheme Portal focuses on improving the internship discovery experience through a recommendation-based system. The platform allows students to explore and apply for internships while providing administrators with tools to manage opportunities and applications efficiently.

The main objective of the project was to create a scalable and user-friendly portal capable of matching students with suitable internship opportunities.
